A library providing a simplified API for the Lightning Dev Kit. While LDK itself provides a highly configurable and adaptable interface, this API champions simplicity and ease of use over configurability. To this end, it provides an opionated set of design choices and ready-to-go default modules, while still enabling some configurability when dearly needed by the user:

  • Chain data is accessed through an Esplora client.
  • Wallet and channel states are persisted to disk.
  • Gossip is retrieved over the P2P network.
